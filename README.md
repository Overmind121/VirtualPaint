# VirtualPaint
## Project Description 
This code uses open cv to make a virtual paint app. The paint app allows the user to draw and change colors using a virtual pallette 

## How to Use
* Create color filter to detect an object that will act as your brush
* Once you specified your color filter, you can then use your object as a brush to paint stufff
* To change colors, hover your "brush" above the rectangles and press the key 'p' to change colors
